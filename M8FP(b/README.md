# Документация бота техподдержки "Продаем все на свете"

Данный бот предназначен для автоматизации работы службы поддержки интернет-магазина. Бот отвечает на частые вопросы (FAQ) и сохраняет заявки пользователей в базу данных для двух отделов: Программисты и Отдел продаж.

## Стек технологий
* **Язык:** Python 3.10+
* **Библиотека:** pyTelegramBotAPI (telebot)
* **База данных:** SQLite 3

## Структура проекта
* `bot.py` — основной файл запуска бота и логика интерфейса.
* `bdtehpodderjka.py` — менеджер базы данных (класс `DB_Manager`), создание таблиц и SQL-запросы.
* `config.py` — конфигурационный файл (токен бота и путь к БД).
* `tehpodderjka.db` — файл базы данных (создается автоматически).

## Инструкция по развертыванию
1. Установите библиотеку: `pip install pyTelegramBotAPI`
2. В файле `config.py` укажите ваш `TOKEN`, полученный от @BotFather.
3. Запустите сначала `bdtehpodderjka.py`, чтобы инициализировать базу данных и загрузить FAQ.
4. Запустите `bot.py` для работы бота.

## Работа с Базой Данных для Администратора
База данных содержит две основные таблицы:
1.  **faq** — хранит вопросы и ответы. Чтобы изменить ответ бота, достаточно обновить запись в этой таблице.
2.  **requests** — хранит все обращения пользователей.

### Поля таблицы `requests`:
* `request_id`: Уникальный номер заявки.
* `user_id`: ID пользователя в Telegram.
* `username`: Никнейм пользователя.
* `department`: Отдел, в который направлен запрос (Программисты/Отдел продаж).
* `issue`: Текст проблемы.
* `status`: Текущий статус (по умолчанию 'открыт').

Для просмотра данных без кода рекомендуется использовать программу **DB Browser for SQLite**.
